HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se define como un lenguaje de marcado, ya que utiliza etiquetas para definir la estructura y el contenido de una página web. Estas etiquetas no se visualizan en la página final, sino que le indican al navegador cómo interpretar y mostrar el contenido.

¿Para qué sirve HTML?

HTML es la base fundamental de todas las páginas web. Permite definir los elementos básicos que componen una página, como:

  • Encabezados: Títulos y subtítulos (<h1>, <h2>, etc.).
  • Párrafos: Bloques de texto (<p>).
  • Imágenes: Elementos multimedia (<img>).
  • Enlaces: Vínculos a otras páginas web (<a>) o recursos externos.
  • Listas: Listas ordenadas (<ol>) y no ordenadas (<ul>).
  • Tablas: Presentaciones de datos en forma tabular (<table>).
  • Formularios: Captura de datos del usuario (<form>).
  • Secciones: Agrupación de contenido (<section>).
  • División: Segmentación de contenido (<div>).

Aspectos relevantes de HTML:

  • Estructura semántica: HTML5 introdujo nuevos elementos semánticos que permiten definir mejor el significado del contenido, como header, nav, article, aside, footer, entre otros.
  • Accesibilidad: Es importante escribir código HTML accesible para que todas las personas, independientemente de sus habilidades o limitaciones, puedan acceder y utilizar el contenido de la página web.
  • Validación: Se recomienda validar el código HTML para asegurarse de que cumple con los estándares y no contiene errores. Existen herramientas en línea y validadores para este fin.
  • Combinación con CSS y JavaScript: HTML se complementa con CSS para darle estilo a la página y con JavaScript para agregar interactividad y dinamismo.

CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se utiliza para dar formato a las páginas web. Se complementa con HTML, que define la estructura y el contenido de la página, y permite controlar la apariencia visual de los elementos HTML, como colores, fuentes, tamaños, márgenes, disposición y mucho más.

¿Para qué sirve CSS?

CSS es esencial para crear páginas web atractivas, organizadas y consistentes. Permite:

  • Aplicar estilos a elementos HTML: Puedes definir estilos para elementos específicos como párrafos, encabezados, imágenes, enlaces, listas, tablas, formularios, etc.
  • Controlar colores, fuentes y tamaños: Define la paleta de colores de tu página, elige las fuentes tipográficas que desees y establece los tamaños de texto, encabezados y otros elementos.
  • Organizar el diseño: Controla el espaciado entre elementos, los márgenes, el relleno, la alineación y la posición de los elementos en la página.
  • Crear diseños responsivos: Adapta la apariencia de la página a diferentes tamaños de pantalla, como dispositivos móviles y tablets.
  • Mejorar la accesibilidad: CSS puede utilizarse para mejorar la accesibilidad de la página web para personas con discapacidades visuales o de otro tipo.

Sintaxis básica de CSS:

La sintaxis básica de CSS se compone de tres partes:

  1. Selector: Define el elemento HTML al que se aplicarán los estilos. Puede ser un tipo de elemento específico (p.ej., p), una clase (p.clase-ejemplo), un ID (#mi-elemento), o combinaciones más complejas.
  2. Propiedad: Define el aspecto que se desea modificar del elemento. Por ejemplo, color, font-family, font-size, margin, padding, width, height, display, etc.
  3. Valor: Define el valor que se asigna a la propiedad. Por ejemplo, red, Arial, 16px, 10px, auto, etc.

Las reglas CSS se escriben en pares de propiedad-valor separados por dos puntos, y las reglas se agrupan dentro de bloques delimitados por llaves {}.

Aspectos relevantes de CSS:

  • Cascadas: Las reglas CSS se aplican en cascada, es decir, las reglas más específicas tienen prioridad sobre las más generales.
  • Selectores avanzados: CSS ofrece una amplia gama de selectores para definir con precisión a qué elementos se aplican los estilos.
  • Modelos de diseño: Existen diferentes modelos de diseño CSS, como el modelo de caja y el modelo de flexbox, que permiten crear diseños complejos y adaptables.
  • Preprocesadores CSS: Existen herramientas como Sass y LESS que facilitan la escritura de código CSS más organizado y mantenible.

JavaScript

JavaScript es un lenguaje de programación de alto nivel interpretado que se utiliza principalmente en páginas web para agregar interactividad y dinamismo. Se ejecuta en el lado del cliente, es decir, en el navegador del usuario, lo que permite que las páginas web respondan a las acciones del usuario, realicen cálculos, muestren contenido dinámico y creen experiencias más atractivas e interactivas.

¿Para qué sirve JavaScript?

JavaScript tiene una amplia gama de aplicaciones en el desarrollo web, incluyendo:

  • Animaciones y efectos interactivos: Crea animaciones, transiciones y efectos visuales para mejorar la experiencia del usuario y hacer que las páginas web sean más dinámicas.
  • Juegos y aplicaciones web: Desarrolla juegos sencillos, aplicaciones web interactivas y herramientas en línea que se ejecutan en el navegador del usuario.
  • Validación de formularios: Valida la información ingresada por los usuarios en formularios, evitando errores y mejorando la seguridad de los datos.
  • Interacción con el usuario: Permite que la página web responda a las acciones del usuario, como clics, movimientos de mouse, eventos del teclado, etc.
  • Carga de contenido dinámico: Carga y actualiza contenido en la página sin necesidad de recargarla completamente, mejorando la fluidez y la experiencia del usuario.
  • Creación de interfaces de usuario interactivas: Desarrolla interfaces de usuario dinámicas y receptivas que se adaptan a diferentes dispositivos y tamaños de pantalla.
  • Acceso a la API del navegador: Permite acceder a las funcionalidades del navegador web, como la cámara, la geolocalización, el almacenamiento local y más.
  • Desarrollo de aplicaciones web de una sola página (SPA): Crea aplicaciones web completas que se ejecutan en una sola página, sin necesidad de cargar varias páginas diferentes.
  • Integración con frameworks JavaScript: Utiliza frameworks populares como React, Angular o Vue.js para facilitar el desarrollo de aplicaciones web complejas y escalables.

Sintaxis básica de JavaScript:

JavaScript utiliza una sintaxis similar a otros lenguajes de programación, con variables, operadores, estructuras de control, funciones y objetos.

  • Variables: Las variables almacenan datos que pueden utilizarse en el programa. Se declaran utilizando la palabra clave var, let o const, seguido del nombre de la variable y el valor inicial (opcional).
  • Operadores: Los operadores realizan operaciones aritméticas, lógicas y de comparación sobre los datos. Algunos operadores comunes son +, -, *, /, %, ==, !=, <, >, <=, >=, &&, ||, !, etc.
  • Estructuras de control: Las estructuras de control permiten controlar el flujo del programa. Algunas estructuras comunes son if-else, switch, for, while, do-while, etc.
  • Funciones: Las funciones son bloques de código reutilizables que realizan una tarea específica. Se definen utilizando la palabra clave function, seguido del nombre de la función, paréntesis (para los parámetros), llaves ({}) y el cuerpo de la función.
  • Objetos: Los objetos son colecciones de datos estructurados que contienen propiedades y métodos. Se declaran utilizando llaves ({}) y las propiedades se definen como nombre_propiedad: valor_propiedad. Los métodos son funciones asociadas al objeto.

Aspectos relevantes de JavaScript:

  • Lenguaje interpretado: JavaScript se interpreta en el momento de la ejecución, lo que significa que no necesita compilarse antes de ejecutarse.
  • Lenguaje dinámico: JavaScript es un lenguaje dinámico, lo que significa que el tipo de dato de una variable puede cambiar durante la ejecución del programa.
  • Lenguaje orientado a prototipos: JavaScript utiliza un modelo de herencia basado en prototipos, lo que permite crear objetos y estructuras de datos complejas.
  • Lenguaje multiparadigma: JavaScript es un lenguaje multiparadigma, lo que significa que combina elementos de la programación imperativa, funcional y orientada a objetos.
  • Lenguaje en constante evolución: JavaScript está en constante evolución, con nuevas características y funcionalidades.
